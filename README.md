# 🎵 ממיר 3GP ל-MP3

ממיר קבצי 3GP/3GPP/AMR ל-MP3, רץ לגמרי בדפדפן (FFmpeg.wasm) - הקבצים לא עוזבים את המכשיר שלך.

## תכונות

- 📱 PWA - ניתן להתקנה על הטלפון
- 🔗 תמיכה ב-Web Share Target - שיתוף קבצים מאפליקציות אחרות
- 🔄 המרה מרובה של מספר קבצים בבת אחת
- 🎧 איכות 192kbps
- 🔒 פרטיות מלאה - הכל קורה במכשיר

## שימוש מקומי

פתח את `index.html` דרך שרת מקומי (FFmpeg.wasm דורש HTTP/HTTPS, לא file://):

```bash
python -m http.server 8000
# או
npx serve
```
